⏰ Digital Clock (Java)

A simple and elegant Digital Clock application developed using Java and Swing.
It displays the current system time in hours, minutes, and 
seconds and updates in real-time with a user-friendly interface.

 🖥️ Features

- Real-time system clock display
- Hours, Minutes, and Seconds in HH:MM:SS format
- Built using Java's Swing GUI toolkit
- Lightweight and responsive design
- Customizable layout and fonts
